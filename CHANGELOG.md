# Changelog

## 2026-02-18 — Renamed: Claudius → Claudicle

The project was renamed from **Claudius** to **Claudicle** (Claude + cuticle)—the body that forms around a soul.

- GitHub repo: `tdimino/claudius` → `tdimino/claudicle`
- All internal references updated (env vars, file names, docs, code)
- Old URLs redirect automatically via GitHub
- Env var prefix: `CLAUDIUS_*` → `CLAUDICLE_*`
- Main daemon: `daemon/claudius.py` → `daemon/claudicle.py`
- Handoff hook: `hooks/claudius-handoff.py` → `hooks/claudicle-handoff.py`
- LaunchD plists: `com.claudius.*` → `com.claudicle.*`
